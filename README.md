## 4.0" HDMI  touchscreen Display(MPI4008) for raspberry Pi 4 + micro hdmi adapter
### Install drivers

```sudo rm -rf LCD-show```<br>
```git clone https://github.com/goodtft/LCD-show.git```<br>
```chmod -R 755 LCD-show```<br>
```cd LCD-show/```<br>
```sudo ./MPI4008-show```<br>
